## 0.1.0 (2025-12-20)

### Features

- First release using release-please
- Pure domain engine with no_std support
- Immutable dots with append-only history
- Explicit ACL snapshots and visibility grants
- Directed, typed links (Followup, Corrects, Supersedes, Related)
- Command → write-set pattern for adapters
- Validation and normalization functions
- Policy enforcement (can_view, can_grant, can_link)

### Documentation

- Comprehensive architecture docs with Mermaid diagrams
- Entity relationship model
- Design decisions and glossary
